## Solution for anagram task
```
Input: 
Arraylist<> having strings (non-repetitive) and could be 1 million in total.

Output:
How many anagrams for each of the given input string.

You are provided a utility function sort(str) which will return the sorted str for the given str
like sortedStr = sort(str);

Example:
abc, 123, 231, 321, bca, xyza, ayzx, yzxa

output:
abc = 1
123 = 2
231 = 2
321 = 2
bca = 1
xyza = 2
ayzx = 2
yzxa = 2
```
